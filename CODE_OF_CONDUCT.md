# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our pledge
We are committed to providing a harassment-free experience for everyone, regardless of background or identity.

## Standards
Be respectful. Assume good faith. Critique ideas, not people. Avoid personal attacks.

## Enforcement
Project maintainers may remove or edit contributions that violate this code of conduct and may restrict participation for repeated violations.

Contributor Covenant: https://www.contributor-covenant.org/version/2/1/code_of_conduct/
